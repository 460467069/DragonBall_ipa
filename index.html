  • 3. How do I play my old account on this new app(How to log in)?
  • For Arena of Saiyan users, the first time you log in, please select Link Account to sign in and enter the link code to enter the old account profile.
  • For users whose accounts are already connected to FB, you can choose both Link Account and Facebook for the first time to log in.
  • For users whose accounts are NOT connected to FB or using Guest account, the first login, you can choose both Link Account and Guest.
  • If you choose Link Account login, you only need to enter Link Code once the first time you sign in. After finding your old account, you can:
  •     1）Bind account to FB(for convenience of changing devices). You can login with both Link Account and FB in the future;
  •     2）Or continue with Link Account login(no Link Code required any more).
  • If you don't remember your Link Code the first time you log in, you can get it by opening Arena of Saiyan or Dragon Ball.

    "Untrusted enterprise developers" solution

    Due to Apple policy, the "Untrusted Enterprise Developer" prompt appears when first installing an Enterprise application on an iOS system


    The solution is as follows (iOS11 system as an example):

  • 1、open the settings in the phone function, select "General"
  • 2、In General, select the "Profile and Device Management" featurevv
  • 3、In the description file and device management in the enterprise application category, select the file name of the enterprise application to be installed (with the prompt when opening), click to enter
  • 4、After entering the enterprise signature page, confirm the company name and application name in the enterprise signature, click the trust "company name of the enterprise signature"
  • 5、back to the desktop, re-open the application to use
